## Lista de Compras
Projeto desenvolvido em Python, com intuito de praticar lógica de programação, manipulação de listas, laços de repetição e formatação de strings. O programa permite criar uma lista de compras de forma simples e organizada.

## Funcionalidades
- O usuário digita os itens separados por vírgula
- O programa exibe cada item numerado automaticamente
- Exibe a lista final formatada de forma legível

## Como rodar
1 - Abra o terminal na pasta 
2 - Execute:

python main.py
